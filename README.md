# Hours Count

**Hours Count** es una aplicación de gestión de relaciones con clientes (CRM) especializada en el seguimiento de tiempos. Ha sido diseñada para permitir que los trabajadores registren de manera precisa y sencilla las horas dedicadas a cada cliente, facilitando la transparencia en la facturación y la gestión de proyectos.

> **Nota sobre el proyecto:** Esta es una versión pública y optimizada para demostración, derivada de un proyecto privado desarrollado para un entorno de producción real. Se han realizado modificaciones para proteger datos sensibles y mejorar la legibilidad del código como parte de mi portafolio profesional.

## 🚀 Funcionalidades Principales

- **Gestión de Clientes:** Panel centralizado para administrar la información de contacto y contratos.
- **Registro de Actividad:** Interfaz intuitiva para que los empleados imputen horas a proyectos específicos.
- **Cálculos Transparentes:** La lógica del sistema está diseñada para ser clara y directa; si el usuario necesita realizar ajustes o divisiones de tiempo, el sistema permite introducir los datos de forma exacta sin procesos ocultos.
- **Reportes de Tiempo:** Visualización de horas totales por trabajador y por cliente.

## 🛠️ Stack Tecnológico

- **Backend:** PHP 
- **Base de Datos:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Gestor de Dependencias:** Composer
- **Arquitectura:** Enfocada en la modularidad y la claridad del código.
